# ELEC 594 Midterm Report

This repository contains the files for the ELEC 594 midterm report.

## Contents

- **midternm_report.txt**: Source files used to compile the report.
- **ELEC_594_Midterm_Report.pdf**: The compiled version of the midterm report.
- **Images**: Figures used in the report.
- **Reference.bib**: Bibliography file containing all references cited in the report.
- **ELEC_594_Midterm_Report.zip**: A complete package of the project downloaded directly from Overleaf, including all LaTeX source files and related resources.

## Notes

The ZIP file was exported directly from Overleaf and contains the full project structure required to compile the report.
